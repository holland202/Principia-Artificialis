## Hypothesis

A personal knowledge base (Obsidian vault) can be transformed from a static collection of notes into a **living, self-organizing information manifold**. By piping raw markdown files through Claude in a continuous retrieval-augmented loop, the system automatically discovers connections, synthesizes ideas, and evolves the geometry of personal knowledge without manual tagging or rigid folder structures.

This is a real-world instantiation of the geometric memory framework from Volume II.

## Core Mechanism

1. **Daily Note Ingestion** → New markdown files perturb the manifold.
2. **Continuous RAG Scan** → Claude reads recent notes against the entire vault.
3. **Geodesic Connection Discovery** → Instead of manual links, the system finds minimum-information paths between ideas.
4. **Synthesis & Crystallization** → New "memory tension" is resolved into higher-order notes or maps.
5. **Self-Expansion** → The knowledge graph grows organically as curvature and resonance update.
