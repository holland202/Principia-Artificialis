# Contributing to Principia Artificialis

Thank you for your interest in contributing!

## Ways to Contribute

- Submit a Research Note
- Propose or work on an Experiment
- Provide mathematical proofs or counterexamples
- Review and critique existing notes
- Add references or literature reviews
- Implement simulations
- Improve documentation

## Pull Requests

PRs are welcome for code, documents, etc. Follow the structure in research_notes/.

## Issues

Use Issues for specific research problems.

## Discussions

Use GitHub Discussions for broader conversations.
